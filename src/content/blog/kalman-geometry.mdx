大多数教材把卡尔曼滤波讲成五个矩阵递推式，背下来能用，但不知道为什么。
换个角度：**KF 本质上是一次正交投影**。

<Theorem type="theorem" name="最优性">
  在线性动态、高斯噪声的假设下，卡尔曼滤波给出的是所有线性无偏估计中方差最小的那一个。
</Theorem>

这个结论的几何解释是：把观测张成的空间当作一个子空间，最优估计就是真值在这个子空间上的投影，
而残差与子空间正交 —— 这正是"新息"（innovation）不携带任何可用信息的含义。

<Collapsible title="一个二维数值例子">
  设状态是二维位置，先验估计 $\hat{x}^- = (0,\, 0)^\top$，先验协方差 $P^- = 4I$。
  观测直接测位置，$H = I$，观测噪声 $R = I$，这一帧测到 $z = (3,\, 1)^\top$。

  卡尔曼增益

  $$
  K = P^-\!\left(P^- + R\right)^{-1} = \tfrac{4}{5}I = 0.8\,I.
  $$

  更新后，状态与协方差同步收缩：

  $$
  \begin{aligned}
  \hat{x} &= \hat{x}^- + K\!\left(z - \hat{x}^-\right) = (2.4,\, 0.8)^\top, \\[2pt]
  P &= (I - K)\,P^- = 0.8\,I.
  \end{aligned}
  $$

  几何上：估计点落在先验 $(0,0)$ 与观测 $(3,1)$ 的连线上、偏向观测 $0.8$ 处 ——
  正是「先验越不确定越信观测」的那一次投影。
</Collapsible>

<Callout type="insight" title="为什么增益长成那样">
  卡尔曼增益 K 的分子是「先验不确定度」，分母是「先验 + 观测不确定度」。
  它其实就是一个信任权重：先验越不确定，就越信观测。
</Callout>

<Collapsible title="完整推导：从最小二乘到递推形式">
  从批处理最小二乘出发，利用矩阵求逆引理把信息矩阵形式转成协方差形式，
  就能得到熟悉的递推式。关键一步是 Sherman–Morrison–Woodbury 恒等式。

  一次更新写成代码就是：

  ```python
  def kf_update(x, P, z, H, R):
      S = H @ P @ H.T + R          # 新息协方差
      K = P @ H.T @ inv(S)         # 卡尔曼增益
      x = x + K @ (z - H @ x)      # 状态更新
      P = (I - K @ H) @ P          # 协方差收缩
      return x, P
  ```
</Collapsible>

<Callout type="warning">
  线性高斯假设在实际的无人机状态估计里几乎从不严格成立。
  EKF 的线性化误差在大角度机动时会显著劣化，这是 UKF / 流形上的滤波存在的原因。
</Callout>
